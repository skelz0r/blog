1. Setup Object Storage on OVH Public Cloud

First, you have to create a new public cloud project in order to use Object Storage.

Follow this link in order to start. You should have the screen below üëá

Enter a project name and a credit card in order to create your project.

When your new project is ready, click on the Object Storage link in the side panel.

Click on the Create an object storage call to action, then:

  1. Choose a location (like Strasbourg (SBG))
  2. Choose a type of container (Private or Public)
  3. Choose a name for your container

3. Configure user access

Back to the OVH Public Cloud interface: you have to create a new user with an access to your private bucket.

You can achieve this by clicking on Users & Roles link in the side panel.

Click on the Create User call to action, then:

  1. Add a description (not really important)
  2. Add at least ObjectStore operator as Role

The new user has been added and should be ready soon. Don‚Äôt forget to note the generated password, which we‚Äôll be used in our configuration (you can generate a new password later by clicking on the three dots at the end of the user line).
